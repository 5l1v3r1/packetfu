Class PacketFu::Read
In: lib/read.rb
Parent: Object

The Read class facilitates reading from libpcap files, which is the native file format for packet capture utilities such as tcpdump, Wireshark, and PacketFu::PcapFile.

Example

  pkt_array = PacketFu::Read.f2a(:file => 'pcaps/my_capture.pcap')

file_to_array() Arguments

  :filename | :file | :out
    The file to read from.
